1. Planning
The project is a Student Management System designed to help schools manage student records efficiently.
It reduces manual record keeping and improves data organization.
The system is intended for school administrators and will be developed as a console-based application using Python.

2. Requirements Analysis
Functional Requirements
Add a student record
View all student records
Search for a student record
Update a student record
Delete a student record
Non-Functional Requirements
Easy to use
Fast and responsive
Reliable and accurate
Runs on any system with Python installed

3. System Design
Student records are stored as dictionaries inside a list.
The system is menu-driven.
Functions used:
addStudent()
viewStudents()
searchStudent()
updateStudent()
deleteStudent()

4. Implementation
The system was implemented using Python.
All functions were written according to the system design and requirements.
The program allows users to add, view, search, update, and delete student records.

5. Testing
The program was tested by:
Adding multiple students
Viewing all student records
Searching for existing and non-existing students
Updating student details
Deleting student records
All features worked correctly.

6. Deployment
The program was saved as student_management.py and uploaded to GitHub.
It can be run on any computer with Python installed.

7. Maintenance
Future improvements may include:
File or database storage
Graphical user interface (GUI)
User authentication
